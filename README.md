**Multi-Agent Movie Intelligence System using LangGraph**
**Project Overview**

This project demonstrates a Multi-Agent AI system where two agents collaborate to analyze movie data and generate insights.

**Agents Used**

1. Data Collector Agent: Collects movie information such as rating and box office data.

2. Analyst Agent: Analyzes the collected data and generates meaningful insights.

**Workflow**

User Input → Collector Agent → Analyst Agent → Final Insight

**Technologies**

> Python

> LangChain

> LangGraph

**Setup Instructions**

1. Clone the repository:

   git clone < your-repository-link >


2. Install dependencies:

   pip install -r requirements.txt


Run the notebook file to see the agent workflow.
