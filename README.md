****Task 1: Multi-Agent Movie Intelligence System using LangGraph****

**Project Overview**

This project demonstrates a Multi-Agent AI system where two agents collaborate to analyze movie data and generate insights.

**Agents Used**

1. Data Collector Agent: Collects movie information such as rating and box office data.

2. Analyst Agent: Analyzes the collected data and generates meaningful insights.

**Workflow**

User Input → Collector Agent → Analyst Agent → Final Insight

**Technologies**

> Python

> LangChain

> LangGraph

**Setup Instructions**

1. Clone the repository:

   git clone < https://github.com/Amitha312005/Soulpage-genai-assignment-amithareddy.git >


2. Install dependencies:

   pip install -r requirements.txt


Run the notebook file to see the agent workflow.

**Example Output**

Box Office: ₹600 Cr

****Task 2: Context Aware Knowledge Bot****

**Project Overview**

The Context Aware Knowledge Bot is an intelligent conversational system that retrieves information from Wikipedia and provides summarized answers.
It maintains conversation memory, allowing it to understand context from previous queries and generate more relevant responses.

**Key Features**

- Interactive command line chatbot

- Wikipedia powered knowledge retrieval

- Conversation memory for contextual understanding

- Automatic summarization of information

- Error handling for unknown queries

**Technologies Used**

Python

Wikipedia API

**How to Run Task 2**
> Step 1

Navigate to the Task 2 folder:

--cd Task2_Context_Aware_Knowledge_Bot

> Step 2

Install dependencies:

--pip install -r requirements.txt

> Step 3

Run the bot:

--python knowledgeBot.py

> Step 4

Start asking questions in the terminal.

Example:

Who is Narendra Modi?

What is Python?


> To stop the bot:

exit
